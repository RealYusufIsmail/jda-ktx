Create a button and apply a listener.

This will use the defaults from ButtonDefaults unless specified as parameters. You cannot configure the component ID, because it will be generated for you.

Return

Button The resulting button instance. You still need to send it with a message!

Parameters

style

The button style.

label

The button label

emoji

The button emoji

disabled

Whether the button is disabled

expiration

The relative expiration time for the listener as Duration, use Duration.INFINITE to disable timeout

user

The user who is authorized to click the button. If the button is pressed by another user it will just defer edit and ignore.

listener

The callback for the button clicks
