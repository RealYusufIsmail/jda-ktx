awaitWith

inline suspend fun <T> Call.awaitWith(throwOnStatus: Boolean = false, crossinline handler: (Response) -> T): T
Content copied to clipboard

Process and await the result of a Call.

This does not handle any HTTP error codes, since it might be of interest to handle them in the handler.

The Response is closed immediately after the handler returns.

Return

The result of the handler.

Parameters

T

The type of the result.

throwOnStatus

Whether to throw an HttpException if the status code is not 2xx.

handler

The result handler, which should process the response fully.
